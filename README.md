## Как устроен detector.py

### Блок 1: Конфигурация
- `GROQ_API_KEY` — ключ для бесплатной LLM
- `TRUSTED_SOURCES` — белый список надёжных доменов
- `SUSPICIOUS_SOURCES` — чёрный список
- `CLICKBAIT_PATTERNS` — regex-паттерны кликбейта

### Блок 2: SourceAnalyzer
Анализирует URL источника:
- Проверяет домен в белом/чёрном списке
- Ищет подозрительные паттерны в домене
- Проверяет HTTPS

### Блок 3: TextAnalyzer
Эвристический анализ текста:
- Поиск кликбейта (regex)
- Эмоциональные манипуляции
- Заглавные буквы, восклицательные знаки
- Теории заговора
- Качество (длина, цитаты, цифры)

### Блок 4: LLMAnalyzer
Анализ нейросетью через Groq API:
- Модель: Llama 3 8B (бесплатно)
- Получает JSON с вероятностью фейка
- Даёт объяснение и рекомендацию

### Блок 5: FakeNewsDetector
Объединяет всё:
- Веса: 35% текст + 25% источник + 40% ИИ
- Выдаёт итоговый рейтинг 0-100

## Бесплатная LLM — Groq

Groq предоставляет БЕСПЛАТНО:
- **14,400 запросов в день**
- Llama 3 8B — быстрая и умная модель
- Mixtral 8x7B — более мощная

Получить ключ:
1. https://console.groq.com
2. Создать аккаунт (можно через Google)
3. API Keys → Create API Key
4. Вставить в `GROQ_API_KEY` в detector.py

## Формула итоговой оценки

```
Если Groq настроен:
  Итог = Текст×35% + Источник×25% + ИИ×40%

Если Groq НЕ настроен:
  Итог = Текст×60% + Источник×40%
```

Шкала доверия:
- 80-100: ✅ ДОСТОВЕРНО
- 65-79:  🟡 ВЕРОЯТНО ДОСТОВЕРНО  
- 50-64:  ⚠️ ТРЕБУЕТ ПРОВЕРКИ
- 35-49:  🚨 ПОДОЗРИТЕЛЬНО
- 0-34:   🔴 ВЕРОЯТНЫЙ ФЕЙК
