# 🌍 Переключатель языков интерфейса

## Что было добавлено

✅ **Выбор языка в навигации** — выпадающее меню в верхней части\
✅ **6 языков поддержки** — Русский, English, Deutsch, Français, Español, 中文\
✅ **Сохранение предпочтений** — выбранный язык сохраняется в браузере\
✅ **Динамическое обновление** — все элементы интерфейса меняются при смене языка

## 🚀 Как использовать

1. **Запусти приложение:**
   ```bash
   python app.py
   ```

2. **Откройте браузер** → `http://localhost:5000`

3. **Выбери язык** в меню справа сверху (выпадающее меню "🌍 Язык:")

## 💫 Как это работает

- **Выбор языка** сохраняется в `localStorage` браузера
- **При каждом анализе** выбранный язык отправляется на сервер
- **API возвращает результаты** на выбранном языке
- **Текущие новости (RSS)** обновляются на выбранном языке

## 🌐 Поддерживаемые языки

| Флаг | Язык | Код |
|------|------|-----|
| 🇷🇺 | Русский | `ru` |
| 🇬🇧 | English | `en` |
| 🇩🇪 | Deutsch | `de` |
| 🇫🇷 | Français | `fr` |
| 🇪🇸 | Español | `es` |
| 🇨🇳 | 中文 | `zh` |

## 📝 Пример

```
1. Откройь приложение
2. Выбери "English (EN)" из меню языков
3. Заголовок изменится на: "Fake News Detector"
4. Все интерфейсные элементы переключатся на английский
5. Результаты анализа будут на английском языке
```
