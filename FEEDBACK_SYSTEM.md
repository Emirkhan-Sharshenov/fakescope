# 📊 Visitor Tracking & Feedback System

Новые возможности для отслеживания посещений и сбора отзывов пользователей.

## 🎯 Функциональность

### 1. **Отслеживание посетителей**
- Автоматически логирует каждого посетителя при входе на сайт
- Хранит: хеш IP (для конфиденциальности), user agent, язык, временную метку
- Данные сохраняются в `data/visitors.jsonl`

### 2. **Система обратной связи**
- Пользователи могут оценить результаты анализа: 👍 (полезно) или 👎 (не полезно)
- Возможность добавить комментарий с объяснением
- Интегрирована на странице результатов анализа
- Работает на всех 6 языках (RU, EN, DE, FR, ES, ZH)

### 3. **Статистика и аналитика**
- Публичный API `/analytics/stats` с общей статистикой
- Отслеживание процента одобрения (approval rate)
- Последние отзывы с комментариями

## 📁 Структура данных

### Файлы хранилища (в папке `data/`)
```
data/
  ├── visitors.jsonl     # Логи посетителей (одна запись = одна строка)
  └── feedback.jsonl     # Отзывы пользователей (одна запись = одна строка)
```

### Формат данных

**Запись о посетителе** (`visitors.jsonl`):
```json
{
  "timestamp": "2024-04-18T10:30:45.123456",
  "ip_hash": "a1b2c3d4e5f6g7h8",
  "user_agent": "Mozilla/5.0...",
  "language": "ru"
}
```

**Отзыв пользователя** (`feedback.jsonl`):
```json
{
  "timestamp": "2024-04-18T10:35:12.654321",
  "ip_hash": "a1b2c3d4e5f6g7h8",
  "analysis_id": "abc123def456",
  "rating": "up",
  "comment": "Полезный анализ!",
  "language": "ru"
}
```

## 🔌 API Endpoints

### POST `/feedback`
Отправить отзыв об анализе

**Параметры:**
```json
{
  "analysis_id": "abc123",      // Хеш анализа (из результатов)
  "rating": "up",               // "up" или "down"
  "comment": "Комментарий...",  // Опционально, макс 500 символов
  "lang": "ru"                  // Язык (для статистики)
}
```

**Ответ:**
```json
{
  "ok": true,
  "message": "✅ Спасибо за отзыв!"
}
```

### GET `/analytics/stats`
Получить статистику

**Ответ:**
```json
{
  "total_visitors": 150,
  "total_feedback": 42,
  "thumbs_up": 35,
  "thumbs_down": 7,
  "approval_rate": 83.3,
  "languages": {
    "ru": 25,
    "en": 15,
    "de": 2
  },
  "recent_comments": [
    {
      "rating": "up",
      "comment": "Отличный анализ!",
      "language": "ru"
    }
  ]
}
```

## 📝 Использование в коде

```python
from analytics import AnalyticsManager

analytics = AnalyticsManager()

# Отследить посетителя
analytics.track_visitor(ip_address, user_agent, lang='ru')

# Сохранить отзыв
success = analytics.save_feedback(
    ip=ip_address,
    analysis_id='abc123',
    rating='up',
    comment='Отличная работа!',
    lang='ru'
)

# Получить статистику
stats = analytics.get_stats()
rate = analytics.get_approval_rate()
```

## 🔐 Конфиденциальность

- ✅ IP-адреса хешируются (SHA256, первые 16 символов)
- ✅ Исходные IP не хранятся
- ✅ User Agent сокращён до 100 символов
- ✅ Комментарии лимитированы на 500 символов
- ✅ Все данные локальные (нет отправки на сторонние сервисы)

## 🎨 UI Компоненты

### Кнопки оценки
- Расположены в нижней части результатов анализа
- Адаптивный дизайн с визуальной обратной связью
- Поддержка всех 6 языков

### Поле комментария
- Опциональное поле для детального отзыва
- Автоматическое управление (активируется при выборе оценки)
- Вводится после нажатия на оценку
